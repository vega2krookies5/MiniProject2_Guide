┌──────────────────────────────────────────────────────┐
│  Header                                              │

├──────────────────────────────────────────────────────┤
│                                                      │

### 2.2 공통 컴포넌트 목록
| 컴포넌트명 | 위치 | 역할 | 항상 표시 |
|------------|------|------|-----------|
| `Header` | 상단 | 로고, 네비게이션, 로그인 상태 | 예 |
| `Footer` | 하단 | 저작권, 링크 | 예 |

예시 (대출 신청):

### 화면 ID: SCR-001 — 도서 목록 페이지 (작성 예시)

#### 3.1.1 화면 개요
| 항목 | 내용 |
|------|------|
| **화면명** | 도서 목록 페이지 |
| **URL** | `/` |
| **화면 타입** | [x] 목록 |
| **권한 레벨** | [x] 공개 |
| **주요 기능** | 도서 검색, 카테고리 필터, 대출 신청 (로그인 필요) |
| **연결 API** | `GET /api/books?keyword=&category=&page=0&size=10` |

#### 3.1.2 와이어프레임 (ASCII)
```
┌──────────────────────────────────────────────────────┐
│ [로고] 도서관   [도서목록] [내대출이력]  [로그인]       │
├──────────────────────────────────────────────────────┤
│                                                      │
│  📚 도서 검색                                         │
│  ┌────────────────────────────────┐ [검색]           │
│  │ 도서명 또는 저자 입력...        │                  │
│  └────────────────────────────────┘                  │
│                                                      │
│  카테고리: [전체▼] [소설] [IT] [역사] [기타]           │
│                                                      │
│  ┌─────────────────────────────────────────────┐    │
│  │ 도서명        │ 저자   │ 카테고리 │ 재고 │ 액션  │    │
│  ├───────────────┼────────┼──────────┼──────┼───── │    │
│  │ 클린 코드     │ 로버트 │ IT       │ ●2권 │[대출]│    │
│  │ 해리포터      │ 롤링   │ 소설     │ ●3권 │[대출]│    │
│  │ 총균쇠        │ 다이아 │ 역사     │ ○0권 │ 매진 │    │
│  └─────────────────────────────────────────────┘    │
│                                                      │
│              [◀] [1] [2] [3] [▶]                    │
└──────────────────────────────────────────────────────┘
```

#### 3.1.3 UI 컴포넌트 매핑
| 화면 요소 | React 컴포넌트 | 라이브러리 컴포넌트 | 비고 |
|-----------|---------------|-------------------|------|
| 검색 입력창 | `BookSearch` | `Input` (shadcn) | Enter 또는 버튼으로 검색 |
| 카테고리 필터 | `CategoryFilter` | `Select` (shadcn) | 전체/소설/IT/역사/기타 |

#### 3.1.4 화면 상태 목록
| 상태 | 표시 내용 | 트리거 |
|------|-----------|--------|
| **로딩 중** | 테이블 Skeleton 표시 | 검색/필터/페이지 변경 |
| **결과 있음** | 도서 목록 테이블 정상 표시 | API 성공, 데이터 1건 이상 |
| **결과 없음** | "검색 결과가 없습니다" | API 성공, 데이터 0건 |
| **에러** | "오류가 발생했습니다. 새로고침 해주세요." | API 실패 |

#### 3.1.5 사용자 인터랙션 흐름
```
[검색어 입력 + 엔터]
    → fetchBooks({ keyword, category, page: 0 })
    → API: GET /api/books?keyword=클린코드&page=0
    → 결과 테이블 갱신

[대출 버튼 클릭] (비로그인)
    → "로그인이 필요합니다" 토스트 + /login 이동

[대출 버튼 클릭] (로그인 상태)
    → LoanDialog 열림 (도서명, 반납예정일 = 오늘+14일 표시)
    → [확인] → POST /api/loans { bookId }
    → 성공: "대출 신청 완료" 토스트 + Dialog 닫힘
    → 실패: "대출 가능 권수 초과" 등 에러 메시지
```

## 4. 네비게이션 흐름도

```
[메인: 도서 목록 /]
    │
    ├── [도서 상세 /books/:id]
    │       └── [대출 신청 Dialog] → 성공 → [내 대출이력 /my/loans]
    │
    ├── [로그인 /login]
    │       └── 성공 → [이전 페이지 또는 /]
    │
    ├── [회원가입 /register]
    │       └── 성공 → [로그인 /login]
    │
    └── [내 대출이력 /my/loans] (로그인 필요)
            └── [반납 신청] → PATCH /api/loans/:id/return
```
