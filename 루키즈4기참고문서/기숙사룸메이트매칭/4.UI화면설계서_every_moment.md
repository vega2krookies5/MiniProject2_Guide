## 화면 구조 설계

### 레이아웃 구성

```
┌───────────────────────────────────────────────┐
│                    header                     │
├───────────────────────────────────────────────┤
│                  Main contents                │
│   ┌─────────────────────────┬───────────────┐ │
│   │        페이지 소개        │    상태 카드   │ │
│   │   - 프로젝트 설명         │  - 설문 현황   │ │
│   │   - 설문조사 이동 버튼     │  - 매칭 현황   │ │
│   └─────────────────────────┴───────────────┘ │
├───────────────────────────────────────────────┤
│                    게시판                      │
│       - 공지 / 자유 / 매칭 / 룸메찾기 링크        │
└───────────────────────────────────────────────┘

```

### 주요 섹션

| 섹션명         | 위치 | 역할                                                                                            | 필수여부 |
| -------------- | ---- | ----------------------------------------------------------------------------------------------- | -------- |
| Header         | 상단 | 로고, 채팅 아이콘 이동(`/chat`), 프로필 이동(`/profile`), 메뉴 드롭다운/로그아웃                | 필수     |
| Content Left   | 중앙 | 타이틀/서브타이틀, 서비스 소개 문구 및 CTA(예: 설문/매칭으로 유도)                              | 필수     |
| Status Card    | 중앙 | 개인 성향에 관한 **설문 완료 여부 및 룸메이트 매칭 상태** 실시간 요약(로딩/완료/대기/없음 표시) | 필수     |
| Boards Section | 하단 | 게시판 바로가기 카드(공지/자유/매칭/룸메 찾기) 링크 네비게이션                                  | 필수     |

## 화면 개요

### 화면 목적

- **주요 기능**: 로그인 한 사용자가 메인 진입 시 **현재 상태 (개인 성향에 관한 설문/룸메이트 매칭)** 를 한눈에 확인
  : 게시판/채팅/프로필 등 핵심 기능으로 빠르게 이동
- **사용자 그룹**: 로그인 한 사용자 (ProtectedRoute로 보호되는 페이지에서 진입)

### 화면 분류

- **화면 타입**: [ ] 목록 화면 [ ] 상세 화면 [ ] 입력 화면 [O] 대시보드 [ ] 기타
- **권한 레벨**: [ ] 공개 [O] 로그인 필요 [ ] 관리자 전용 [ ] 특정 권한 필요
